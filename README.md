# Python Fundamentals — AIML Track

A structured collection of Python fundamentals exercises and assignments designed for learners transitioning into **AI/ML & Data Science**. Each file covers a core concept with clear comments, examples, and practice problems.

---

## 📁 Project Structure

```
Python-AIML/
│
├── Python Fundamentals (Part 1)/
│   ├── 1_variables.py                          # Variables & data types
│   ├── 2_input.py                              # User input & type conversion
│   ├── 3_average.py                            # Float input & averages
│   └── Python Fundamentals (Part 1).py         # Assignment 1 (10 problems)
│
└── Python Fundamentals (Part 2)/
    ├── 1_conditional_statement.py              # if-else conditionals
    ├── 2_match_case_conditional_statement.py   # match-case (pattern matching)
    ├── 3_while_loop.py                         # while loops, break & continue
    ├── 4_for_loop.py                           # for loops & range()
    ├── 5_functions.py                          # Functions, parameters & defaults
    ├── 6_types_of_functions.py                 # 8 types of functions (lambda, recursion, etc.)
    └── Python Fundamentals (Part 2).py         # Assignment 2 (10 problems)
```

### Part 1: Basics

## 📖 Learning Tips

- **Uncomment & experiment**: Most code is commented out as templates — uncomment, modify, and run
- **Read the comments**: Each file has detailed explanations and hints
- **Solve assignments**: Try writing solutions before looking at commented answers
- **Break things on purpose**: Change values, remove lines, see what errors you get — it's the fastest way to learn
